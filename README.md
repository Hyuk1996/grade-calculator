# grade-calculator

<br/>

## 프로젝트 개발 이유

JAVA를 공부하면서 객체지향적으로 프로그래밈을 만드는 법을 연습하기 위해서 시작한 프로젝트입니다.

<br/>

<br/>

## 프로젝트 내용

대학생들의 성적을 입력 받아, 학점을 계산해주는 프로그램입니다. 

<br/>

<br/>

## 프로젝트 구성

School class

Student class

Subject class

Score class
